# Changelog

All notable changes are documented here.

## 0.1.0 - 2026-07-18

- Published the project and installable command as 8bit Buddy (`8bit-buddy`).
- Added Codex, Claude Code, and Cursor lifecycle adapters.
- Added concurrent main-agent and subagent state tracking.
- Added AWTRIX 3 custom-app output for the Ulanzi TC001.
- Added conservative attention detection and explicit permission handling.
- Added safe, idempotent hook installation with timestamped backups.
- Added a macOS LaunchAgent installer, health checks, demo mode, and CLI status.
