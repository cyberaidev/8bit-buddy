# Security policy

## Supported versions

Until the first stable release, security fixes are applied to the latest `0.x` version only.

## Reporting a vulnerability

Do not open a public issue for a vulnerability. Use GitHub's **Report a vulnerability** option in the
Security tab to create a private advisory. Include affected versions, impact, reproduction steps, and
any proposed mitigation. Please allow maintainers reasonable time to investigate before disclosure.

## Scope reminders

8bit Buddy should remain bound to loopback. Exposing its API or forwarding agent events across a
network is unsupported unless an authenticated and encrypted transport is added.
